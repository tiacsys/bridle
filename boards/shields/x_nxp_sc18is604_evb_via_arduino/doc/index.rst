NXP SC18IS604_EVB Interconnection shield
########################################

.. toctree::

This shield does not correspond to a conventional plug-on module, but to the
wiring for connecting required signals from a host board to the
:ref:`nxp_sc18is604_evb_shield` via the |Arduino UNO R3| connector standard.
