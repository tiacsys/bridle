CANnectivity
############

The `CANnectivity Project`_ is an open source firmware for Universal Serial
Bus (USB) to Controller Area Network (CAN) adapters. The firmware, which is
based on the `Zephyr RTOS <Zephyr Project_>`_, allows turning your favorite
microcontroller development board into a full-fledged USB to CAN adapter.

The |BRIDLE| refere to the original CANnectivity Project, read the Bridle
:ref:`getting_started`, the CANnectivity :external+cannectivity:doc:`building`
and :external+cannectivity:doc:`porting`.
